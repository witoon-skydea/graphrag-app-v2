# โครงการ GraphRAG: ระบบจัดการความรู้เชิงกราฟสำหรับองค์กร

## ภาพรวมโครงการ

GraphRAG เป็นโปรเจคพัฒนาระบบ Retrieval-Augmented Generation ขั้นสูงที่ผสานเทคโนโลยี Knowledge Graph และ Vector Database เข้าด้วยกัน เพื่อเพิ่มประสิทธิภาพในการค้นหาและเชื่อมโยงข้อมูลภายในองค์กรแบบอัตโนมัติ โครงการนี้เป็นความร่วมมือระหว่างทีมนักวิจัยของ SkyTech AI และมหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรี

ระบบ GraphRAG ได้รับทุนสนับสนุนบางส่วนจากสำนักงานพัฒนาวิทยาศาสตร์และเทคโนโลยีแห่งชาติ (สวทช.) ภายใต้โครงการส่งเสริมการวิจัยและพัฒนาเทคโนโลยี AI ในประเทศไทย

## องค์ประกอบหลักของระบบ

1. **Vector Database Integration**
   - รองรับการเชื่อมต่อกับ Weaviate, Pinecone และ Qdrant
   - สร้าง Embedding ด้วย Ollama (Local) หรือ OpenAI Embedding API
   - รองรับการค้นหาแบบ Semantic Search และ Hybrid Search

2. **Knowledge Graph Builder**
   - สกัด Entities จากเอกสารด้วย NER หรือ LLM
   - วิเคราะห์ความสัมพันธ์ระหว่าง Entities
   - สร้างและจัดการ Knowledge Graph
   - รองรับการวิเคราะห์ความสัมพันธ์ข้ามเอกสาร

3. **Document Processing Pipeline**
   - ประมวลผลเอกสารหลากหลายรูปแบบ (TXT, PDF, DOCX, CSV)
   - สกัดข้อความจากรูปภาพด้วย OCR
   - แบ่งเอกสารเป็นชิ้นส่วนเพื่อการค้นหาที่แม่นยำ

4. **Query Processing Module**
   - แปลงคำถามเป็นการค้นหาใน Vector DB และ Knowledge Graph
   - ผสานผลลัพธ์จากทั้งสองแหล่งข้อมูล
   - สร้างคำตอบโดยใช้ LLM กับข้อมูลที่ได้

## ทีมงานหลัก

- **หัวหน้าโครงการ**: นายสมชาย ใจดี (หัวหน้าทีมพัฒนา, SkyTech AI)
- **ที่ปรึกษาด้าน AI**: ดร.พิชัย นวัตกรรม (CEO, SkyTech AI)
- **นักวิทยาศาสตร์ข้อมูลอาวุโส**: นางสาวรัตนา วิทยา (SkyTech AI)
- **วิศวกรซอฟต์แวร์อาวุโส**: นายอนุชิต รุ่งเรือง (SkyTech AI)
- **อาจารย์ที่ปรึกษา**: ผศ.ดร.สุพจน์ เตชวรสินสกุล (มจธ.)
- **นักศึกษาปริญญาโท**: นายวิชิต ปัญญาดี (มจธ.)

## ความก้าวหน้าและแผนการดำเนินงาน

โครงการเริ่มต้นเมื่อเดือนมกราคม 2025 และมีกำหนดแล้วเสร็จในเดือนธันวาคม 2025 ปัจจุบันโครงการอยู่ในระยะที่ 2 จากทั้งหมด 4 ระยะ

**ระยะที่ 1**: การพัฒนาโมดูล Document Processing และ Vector Database (แล้วเสร็จ)
**ระยะที่ 2**: การพัฒนาโมดูล Knowledge Graph Builder (กำลังดำเนินการ)
**ระยะที่ 3**: การพัฒนาโมดูล Query Processing และการบูรณาการระบบ (กำหนดเริ่มเดือนกรกฎาคม 2025)
**ระยะที่ 4**: การพัฒนา Web UI และการทดสอบกับผู้ใช้งานจริง (กำหนดเริ่มเดือนตุลาคม 2025)

## ผลที่คาดว่าจะได้รับ

1. ระบบ GraphRAG ที่สามารถติดตั้งและใช้งานได้ในองค์กรทั่วไป
2. ต้นแบบการบูรณาการระหว่าง Vector Database และ Knowledge Graph
3. ผลงานวิจัยที่สามารถตีพิมพ์ในวารสารวิชาการระดับนานาชาติ
4. เทคโนโลยีด้าน AI สัญชาติไทยที่สามารถแข่งขันในระดับสากล
